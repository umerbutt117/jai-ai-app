Setup Guide - 3 Easy Steps

Step 1: Download JAI AI Code

Download the JAI_AI_COMPLETE_SYSTEM.jsx file from the download link provided.

# Save the file in your project folder JAI_AI_COMPLETE_SYSTEM.jsx

Step 2: Create React App

If you don't have a React app, create one using Vite (fastest):

npm create vite@latest jai-ai-app -- --template react cd jai-ai-app npm install

Step 3: Add JAI AI Component

Replace your App.jsx with the JAI AI code:

# Copy the JAI_AI_COMPLETE_SYSTEM.jsx content # Paste it into: src/App.jsx # Save and run: npm run dev # Your JAI AI will be live on: http://localhost:5173

Step 4: Deploy Free (Vercel)

Deploy for free on Vercel in 2 minutes:

# Install Vercel CLI npm install -g vercel # Deploy vercel # Follow the prompts - your app is live!

Optional: Connect to Real AI (Anthropic API)

For full AI capabilities, add your Anthropic API key:

1. Get API key from: https://console.anthropic.com 2. Add to your .env file: VITE_ANTHROPIC_KEY=your_key_here 3. Update the response logic in JAI_AI_COMPLETE_SYSTEM.jsx 4. Redeploy!
